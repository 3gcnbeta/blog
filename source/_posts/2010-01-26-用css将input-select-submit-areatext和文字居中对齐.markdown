<p>假设HTML上有一个容器id<br />
如果你想把他里面的各种TAG都居中对齐的话，你只需要在<br />
CSS里加入下面的语句</p>
<p>#id *<br />
{<br />
vertical-align: middle;<br />
}</p>
<p>如果你只想指定几样对齐，将*号替换成相应的几个TAG就可以了。</p>
<p>一定要注意id所对应的容易一定是在你想要对齐的那些控件的外面的一个容器。</p>
