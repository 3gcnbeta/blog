<p>1。删除前后的空白字符<br />
>>> str = " a b c d "<br />
>>> str.strip()<br />
'a b c d'<br />
2。删除左边的空白<br />
>>> str = "                a b c d                  "<br />
>>> str.lstrip()<br />
'a b c d                  '<br />
3。删除右边的空白<br />
>>> str = "                a b c d                  "<br />
>>> str.rstrip()<br />
'                a b c d'</p>
