<p>在实现Elixir下OneToOne关系时，<br />
一开始使用的形式是:<br />
class A():<br />
  b = OneToOne('B', inverse='a')</p>
<p>class B():<br />
  a = OneTOOne('A', inverse='b')</p>
<p>发现编译一直无法通过。</p>
<p>然后使用：<br />
class A():<br />
  b = OneToOne('B', inverse='a')</p>
<p>class B():<br />
  a = ManyToOne('A')</p>
<p>后通过.<br />
这里的ManyToOne实际上跟OneToOne的效果是一样的。</p>
